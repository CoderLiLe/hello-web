## 🚀 立即使用

### 本地开发
```bash
# 克隆项目
git clone https://github.com/CoderLiLe/hello-web.git
cd hello-web

# 安装依赖
npm install

# 启动开发服务器
npm run dev
# 访问 http://localhost:8080
```

### 常用命令
```bash
# 开发命令
npm run dev          # 启动开发服务器
npm run build        # 构建生产版本
npm run serve        # 预览构建结果

# 优化命令
npm run optimize:docs  # 优化文档
npm run optimize:code  # 优化代码
npm run optimize:all   # 优化所有内容

# 检查命令
npm run lint:md      # 检查Markdown
npm run format:md    # 格式化文档
npm run check        # 运行所有检查

# 部署命令
npm run commit       # 自动提交代码
npm run deploy       # 构建并提交
```
