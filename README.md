## 🚀 快速开始

### 本地开发

1. 克隆项目
```bash
git clone https://github.com/CoderLiLe/hello-web.git
cd hello-web
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```
