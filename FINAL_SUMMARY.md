## 🔧 使用指南

### 快速开始
```bash
# 1. 克隆项目
git clone https://github.com/CoderLiLe/hello-web.git
cd hello-web

# 2. 安装依赖
npm install

# 3. 启动开发
npm run dev

# 4. 访问文档
# 打开 http://localhost:8080
```

### 常用命令
```bash
# 开发命令
npm run dev          # 启动开发服务器
npm run build        # 构建生产版本
npm run serve        # 预览构建结果

# 优化命令
npm run optimize:docs  # 优化文档
npm run optimize:code  # 优化代码
npm run optimize:all   # 优化所有内容
npm run update         # 完整更新流程

# 检查命令
npm run lint:md        # 检查Markdown格式
npm run format:md      # 格式化代码
```

### 自动化工作流
1. **代码提交**: 自动触发格式化和检查
2. **Pull Request**: 自动运行CI检查
3. **合并到main**: 自动构建和部署到GitHub Pages
